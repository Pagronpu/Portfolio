Final Year Project: Detecting ChatGPT


A research-based project to identify factors that can affect a ChatGPT detector's accuracy (GPTZero). The goal was to see if we could "fool" the detector by using different prompts.

Developed a software testing tool to facilitate the pipeline process using Python.

Utilized prompt engineering to reduce the accuracy of GPTZero using from 92% to 2%.
